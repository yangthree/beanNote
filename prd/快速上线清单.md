# ⚡ 快速上线清单

## 🎯 3步切换到生产环境

### 第1步：获取生产环境 ID

1. 打开微信开发者工具
2. 点击 **"云开发"** → **"环境设置"**
3. 如果没有生产环境，点击 **"创建新环境"**
4. 复制生产环境的 **环境 ID**（例如：`prod-xxxxx`）

### 第2步：修改配置文件

打开 `utils/env.js`，修改两处：

```javascript
const ENV_CONFIG = {
  current: 'prod', // ✅ 改为 'prod'
  envIds: {
    dev: 'test-3g3ho1oo318ec768',
    prod: 'prod-你的实际环境ID' // ✅ 替换为实际的生产环境 ID
  }
}
```

### 第3步：上传云函数

在微信开发者工具中：

1. 右键 `cloudfunctions/publishRecord` → **"上传并部署：云端安装依赖"** → 选择 **生产环境**
2. 右键 `cloudfunctions/getDiscoverList` → **"上传并部署：云端安装依赖"** → 选择 **生产环境**
3. 右键 `cloudfunctions/login` → **"上传并部署：云端安装依赖"** → 选择 **生产环境**

## ✅ 验证

1. 重新编译小程序
2. 查看控制台，应该看到：`[环境配置] 当前环境: 生产环境`
3. 测试核心功能：
   - [ ] 登录
   - [ ] 发布豆单
   - [ ] 查看发现页

## 📤 提交审核

1. 点击 **"上传"** 按钮
2. 填写版本号和备注
3. 在微信公众平台提交审核
